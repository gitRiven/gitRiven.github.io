我的VSCode设置

VSCode安装的插件（常用）

通用插件

  • Auto Close Tag（自动补全标签结尾</>）
  • Auto Rename Tag（自动同步修改标签）
  • AutoFileName（引入文件提示）
  • background（背景）
  • Beautify（代码美化插件）
  • Beacket Pair Colorizer（让括号拥有独立的颜色）
  • Code Runner（运行选中代码段，支持大量语言，包括Node）
  • Dracula Official（黑色主题）
  • EaseServer（开启本地服务器）
  • hightlight-matching-tag（语法高亮）
  • open in browser（在浏览器中打开）
  • Simple icon theme（图标主题）
  • Terminal（可以在文本中直接使用终端命令）

HTML插件

  • HTML CSS Support（标签样式智能提示）
  • HTML Snippets（初级代码片段，自动输入标签：i+tab）

Javascript插件

  • ESLint（语法错误检测）
  • JQuery Code Snippets（初级代码片段）

CSS插件

  • Beautify css/sass/scss/less（美化工具）
  • Sass（Sass代码补全，高亮，语法检测）

PHP插件

  • PHP Debug（搭建PHP环境）
  • PHP Intelephense
  • PHP InTelliSense

Node.js插件

  • Node.js docs（可以打开Node.js文档）

Vue.js插件

  • Vetur（Vue代码补全，高亮，语法检测）
  • Vue 2 Snippets（Vue2代码补全，高亮，语法检测）
